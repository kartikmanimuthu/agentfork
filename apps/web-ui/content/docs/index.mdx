Chatbot is an open-source AI agent and chatbot platform with multi-tenant isolation, RBAC, audit logging, RAG support, and an Agent Studio. Built on Next.js 15, AWS Bedrock, and PostgreSQL with pgvector.

## Features

| Feature | Description |
|---------|-------------|
| Agent Studio | Build, version, and publish AI agents with a visual playground |
| Knowledge Bases | Upload documents, sync web sources, semantic search with pgvector |
| MCP Servers | Connect external tools and APIs via Model Context Protocol |
| LLM Providers | Multi-provider model management (Bedrock, OpenAI-compatible) |
| RAG Pipeline | Retrieval-augmented generation with vector embeddings |
| Analytics & Sessions | Track inference usage, token costs, session history |
| Multi-Tenant | Isolated organizations with tenant-scoped data |
| RBAC | Role-based access control (Owner, Admin, Member, Viewer) |
| Audit Logs | Complete activity trail with filtering |
| Background Jobs | pg-boss workers for embeddings and summaries |
| Auth | NextAuth with Cognito SSO support |
| Chat Widget SDK | Embeddable chat widget for external apps |
