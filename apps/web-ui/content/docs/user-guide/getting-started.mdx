## Prerequisites

- Node.js 20+ or Bun 1.2+
- PostgreSQL 16 with pgvector extension
- AWS account with Bedrock access (Claude models enabled)

## Step 1: Clone and Install

```bash
git clone https://github.com/kartikmanimuthu/chatbot.git
cd chatbot
bun install
```

## Step 2: Configure Environment

Copy the example environment file and fill in your values:

```bash
cp .env.example .env
```

Key variables:
- `DATABASE_URL` — PostgreSQL connection string
- `AWS_REGION` — AWS region with Bedrock access
- `NEXTAUTH_SECRET` — Random secret for session encryption
- `NEXTAUTH_URL` — Your app URL (e.g., `http://localhost:3001`)

## Step 3: Set Up the Database

```bash
bun run prepare
npx prisma migrate deploy
```

## Step 4: Start Development

```bash
bun run dev:all
```

This starts both the web UI (port 3001) and the workers server.

## Step 5: Create Your First Organization

1. Visit `http://localhost:3001`
2. Click **Get Started** and register an account
3. Create your first organization (tenant)
4. You'll be redirected to the dashboard

## Step 6: Create Your First Agent

1. Go to **Agents** in the sidebar
2. Click **New Agent**
3. Enter a name and system prompt
4. Select a model (default: Claude Sonnet via Bedrock)
5. Click **Save**
6. Open the **Playground** to test your agent with live streaming
