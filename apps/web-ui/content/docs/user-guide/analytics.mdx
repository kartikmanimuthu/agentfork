The Analytics module gives you full visibility into your platform's AI operations — from individual inference calls to aggregate cost trends. Use it to monitor usage, control costs, and audit activity across your organization.

## Dashboard Overview

The main Analytics dashboard shows:
- **Total inferences** — count of all inference calls
- **Token usage** — input and output tokens consumed
- **Estimated cost** — calculated from token counts and model pricing
- **Active sessions** — currently open inference sessions
- **Top agents** — most-used agents by inference count

## Filtering

Use the filter bar to narrow results:
- **Date range** — preset or custom range
- **Agent** — filter by specific agent
- **User** — filter by team member
- **Model** — filter by LLM model used

## Inferences List

The **Inferences** page shows a filterable table of all inference calls:
- Status (success, error, in-progress)
- Model used
- Token count (input/output)
- Latency
- Estimated cost
- Timestamp

Click any row to view the full inference detail with messages and metadata.

## Audit Logs

The **Audit** page shows a complete activity trail:
- Event type (login, create, update, delete, etc.)
- Severity (info, warning, error)
- Actor (user or system)
- Timestamp
- Filter by event type, severity, and date range
