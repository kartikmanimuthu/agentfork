# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| < 1.0   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please report it
responsibly via GitHub Security Advisories:

**[Report a Vulnerability](https://github.com/kartikmanimuthu/agentfork/security/advisories/new)**

Please do not open public issues for security vulnerabilities.

We aim to respond to security reports within 48 hours and will keep you
informed of our progress toward a fix. Once resolved, we will publish a
security advisory and credit you (unless you prefer to remain anonymous).

Thank you for helping keep this project and its users safe.
